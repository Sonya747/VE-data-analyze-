# VE-data-analyze 新能源汽车数据分析和可视化

### 文件说明

- visul-pro.py: 可视化三种汽车和新能源汽车产销
- corrilation.py: VE销量和相关仲裁案件数量、油价、车价、充电桩数量、汽车公司平均专利数量的皮尔逊相关指数；相关度热力图
- regression.py: 线性回归、岭回归、Lossa回归建模，销量和corrilation.py中的变量关系，调整参数，可视化拟合效果。
